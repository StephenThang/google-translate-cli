# Google Translate CLI

Command-line translation tool built with Python and deep-translator.

## Setup
pip install -r requirements.txt
python translator.py

## Usage
Enter a source language (name or code), the text to translate,
and a destination language. Supports 100+ languages.

## Features
- Translate between 100+ languages using codes or full names
- Accepts both language codes (en) or full names (english)
- Clean error handling with supported language list on bad input
